# Resonance — Project Specification

## Overview
Resonance is a dating application that uses conversational AI to understand users deeply and match them with compatible people. The product focuses on meaningful connections over superficial swiping.

## Product Vision
- No swiping, no forms
- AI-driven conversations that understand personality, values, and preferences
- City-by-city launch, starting with select colleges
- Genuine compatibility matching based on deep understanding

## Current State
- Public landing page (in development)
- Waitlist collection system
- Single-page design with hero video + waitlist form

## Tech Stack
- HTML5 / CSS3 / Vanilla JavaScript
- Static site (no framework)
- Local development via `live-server`

## Landing Page Sections
1. **Navigation** — Brand name (left) + Join Waitlist CTA (right)
2. **Hero** — Full-viewport video background with headline + description + CTA
3. **Waitlist** — Signup form with name, email, college/city fields
4. **Footer** — Minimal branding + copyright

## Design Direction
- Warm, cinematic, vintage aesthetic
- Color palette: creams, beiges, earthy browns, gold accents
- Typography: Playfair Display (serif) + Inter (sans-serif)
- Subtle animations and film-grain textures
